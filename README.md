# 🛒 Amazon Clone

A responsive Amazon-inspired e-commerce homepage built using HTML, CSS.

## ✨ Features
- Responsive Navbar
- Search Bar
- Hero Section
- Product Cards
- Footer Section
- Mobile-Friendly Design

## 🛠️ Tech Stack
- HTML5
- CSS3

## 🚀 Run Locally
1. Clone the repository
2. Open `Amazon clone.html` in your browser
